== Description ==

The goal of the plugin is to manage your shop without opening the orders.

* You have all info to pack, ship and complete the order.
* See products, variation properties, product counts and total count.
* The shipping address
* Customer comments
* Product costs, shipping method, shipping costs and the total costs

== Installation ==

= 3 ways =
* Use the plugin upload functionality to upload the zip and install the plugin.
* Extract to contents of the zip to the plugins directory of your installation.
* Find the plugin and press install (WordPress plugins)


== Frequently Asked Questions ==

= What if you do not see all columns? =

On the top of the admin order screen there is a little tab 'Screen Options' (WordPress default). You can select columns there and set the number of orders.

= Does it work on WooCommerce 3.3.3+? =

Yes, we tested it with the new order screen.

= Is it compatible with other plugins? =

The WooCommerce structure is not altered. Plugins depending on that should work fine.

= What if I have problems? =

If you deactivate the plugin all effects stop, and plugin data is removed.
